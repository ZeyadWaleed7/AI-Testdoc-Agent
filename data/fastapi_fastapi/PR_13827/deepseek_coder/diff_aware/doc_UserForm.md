## `UserForm` class in Python ##  
The following is a custom data model (class) used to represent user information, which includes name and email fields of type string only for this purpose as per the requirement. The UserForm object can be created by providing two parameters: 'name' with its value being a str representing an individual’s full or preferred username; ‘email', also requiring it to hold valid e-mail addresses in order to function properly, and must contain at least one character before @ symbol followed immediately after the domain name.
